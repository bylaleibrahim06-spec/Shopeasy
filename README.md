## Installation

1. Cloner le dépôt dans le dossier `htdocs`
2. Créer la base de données MySQL `essaibdd`
3. Importer le fichier SQL
4. Configurer `config/connexion.php` avec vos identifiants
5. Accéder via `http://localhost/Shopeasy/accueil.php`
